# 网页章节结构重构 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把章节从「精选（主题）+ Region A–F（地理）」混编，改成 **第一章＝最热门景点，其后按地区依次成章**，并删掉重复的「精华速览」横滑卡与空壳区域章。

**Architecture:** 静态站点生成器改动。核心在 `webpage/build.py`：用一个统一的 `spot_chapters()` 章节模型（第一章 = `HIGHLIGHTS` 成员，其后 = `REGIONS` 中仍有非热门景点的地区）同时驱动正文 `chapters_html()` 与 hero 导航，保证一一对应；空区域自动不成章；章名 eyebrow 改「第 N 章」。`content.py` 小改（`HIGHLIGHTS` 简化为 id 列表、去掉 `B2` 的 highlight 标记）。`style.css` 删除随横滑卡下线而变死的样式。不改投票/云同步/图片/依赖。

**Tech Stack:** Python 3 标准库（`build.py`），原生 CSS，Node 内置 test runner（回归用）。

对应 spec：`docs/superpowers/specs/2026-07-12-章节结构重构-design.md`。

### Task 1: 章节结构重构（content.py + build.py + style.css）

**Files:**
- Modify: `webpage/content.py`（`HIGHLIGHTS` 定义；`B2` item 的 highlight 标记）
- Modify: `webpage/build.py`（`HL_IDS`；删 `highlights()`；`highlights_main()`+`region()` → 章节模型；`hero()` 导航；`BODY`）
- Modify: `webpage/style.css`（删横滑卡样式块）
- Generated（不手改，靠 `build.py` 生成）: `webpage/index.html`

**Interfaces（本任务在 `build.py` 内新增/替换的函数，供正文与导航共用）:**
- `cn_chapter(n:int) -> str`：`1 -> "第一章"`。
- `spot_chapters() -> list[dict]`：有序景点章**元数据**（不渲染 body）。每个 dict 键：`anchor:str`（锚点 id）、`nav:str`（导航短名）、`title:str`（章名 h2）、`data:str`（chap-data 的 HTML 片段，已 esc）、`desc:str`（章描述，可空）、`items:list`（该章的 item 列表）。第一项恒为第一章「最热门景点」（`HIGHLIGHTS` 成员，按序）；其后为 `REGIONS` 中 `items` 非空的地区。
- `render_chapter(n:int, ch:dict) -> str`：把一个章 dict 渲染成 `<section class="chapter">…`，eyebrow 用 `cn_chapter(n)`，body 由 `ch["items"]` 逐个 `spot()` 拼出。
- `chapters_html() -> str`：`enumerate(spot_chapters(), 1)` 渲染全部景点章并拼接。
- `hero()`：导航条改由 `spot_chapters()` 生成（`<b>第N章</b><span>nav</span>`，锚点 `#{anchor}`）。

替换为：

```python
# ---------- 章节：第一章 最热门 + 各非空地区章 ----------
CN_NUM = "一二三四五六七八九十"
def cn_chapter(n):
    """1 -> '第一章'（章数不会超过 10）。"""
    return f"第{CN_NUM[n-1]}章"

def spot_chapters():
    """有序景点章的元数据（不渲染 body，供正文与 hero 导航共用，保证一一对应）。
    第一章＝HIGHLIGHTS（最热门）；其后＝REGIONS 中仍有非热门景点的地区，按原顺序。
    整区景点全为热门的地区（科莫多/布罗莫）items 为空 → 不成章。"""
    chapters = [{
        "anchor": "hl-main", "nav": "最热门", "title": "最热门景点",
        "data": '<span>多数人必去</span>', "desc": "",
        "items": sorted((it for it in C.ITEMS if it["id"] in HL_IDS),
                        key=lambda it: HL_IDS.index(it["id"])),
    }]
    for r in C.REGIONS:
        items = [it for it in C.ITEMS if it["region"] == r["id"] and it["id"] not in HL_IDS]
        if not items:
            continue
        chapters.append({
            "anchor": f'region-{r["id"]}', "nav": r["name"].split(" (")[0], "title": r["name"],
            "data": f'<span>{esc(r["tag"])}</span><i></i><span>{esc(r["days"])}</span><i></i><span>{esc(r["budget"])}</span>',
            "desc": r["desc"], "items": items,
        })
    return chapters

def render_chapter(n, ch):
    desc = f'<p class="chap-desc">{esc(ch["desc"])}</p>' if ch["desc"] else ''
    body = "".join(spot(it) for it in ch["items"])
    return f'''<section class="chapter" id="{ch['anchor']}">
  <div class="chap-head">
    <div class="chap-meta"><h2>{esc(ch['title'])}</h2><span class="chap-region">{cn_chapter(n)}</span></div>
    <div class="chap-data">{ch['data']}</div>
  </div>
  {desc}{body}
</section>'''

def chapters_html():
    return "".join(render_chapter(i, ch) for i, ch in enumerate(spot_chapters(), 1))
```
