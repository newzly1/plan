## 排期硬逻辑（围绕已购机票）

1. 两个已购锚点定死结构：10.2 凌晨 DPS 进、10.10 15:00 LBJ 出 → **巴厘岛前置（10.2–10.6）→ 10.7 DPS→LBJ → 科摩多收尾（10.7–10.10）**。
2. 海上活动最晚排到 10.9（傍晚回港），距 10.10 15:00 航班有整晚缓冲；10.10 上午只留镇上轻活动。
3. **分开票衔接必须留整晚缓冲**（LBJ 小机场延误/取消不罕见，误机无赔付）。此规则现在只约束 LJ：若 10.8 从 DPS 回国，最晚 **10.7 傍晚必须已回到巴厘岛**。

## 各地游玩方式（轻松版）

总原则：**按住宿片区就近玩，一天只排一个主活动**；6 人包一辆车（巴厘岛路窄常堵，跨片区一天跑多点最累）。

| 景点/活动 | 说明 |

| **乌鲁瓦图断崖庙 + Kecak 火舞** ★ | 巴厘岛名片：断崖落日 + 露天火舞表演（约 18:00 场），下午出发即可，排 10.6 |

### 科摩多（拉布安巴焦，10.7–10.10）

住镇上海景酒店，玩法 = 出海 + 镇上晃：

- **出海①经典线**：选**带遮阳棚的一日快艇团**（勿选慢船，海上时间翻倍）；帕达尔岛登顶台阶 30–40 分钟、爬到第二观景台即可出片 → 科莫多龙跟向导平地步道 → 粉沙滩浮潜 → Manta Point 看魔鬼鱼。备晕船药。
- **出海②轻松浮潜线**：Kelor 小岛、Kanawa、Taka Makassar 白沙洲，强度低、以泡水为主。
- **镇上不出海时段**：傍晚 Bukit Cinta 观景台看日落；日落船看**飞狐岛（Kalong）百万蝙蝠出巢**；Kampung Ujung 海鲜夜市；镜石洞 Batu Cermin（镇边，1 小时）。Rangko 海水洞需半天，本行程只能作出海②的替换项。
